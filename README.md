# QUMO

## Campus, Unified.

QUMO is a unified campus operating system designed for student communities, clubs, and educational institutions. It brings together events, quizzes, attendance, announcements, certificates, and resources into a single, easy-to-use platform.

Instead of managing campus activities through multiple tools like Google Forms, WhatsApp groups, Excel sheets, and separate quiz platforms, QUMO simplifies everything into one centralized system.

---

## 🚀 Problem Statement

Modern college communities face fragmented management systems:

- Event registrations via Google Forms
- Attendance tracking on paper or spreadsheets
- Announcements lost in WhatsApp groups
- Separate platforms for quizzes and competitions
- Manual certificate generation
- No proper tracking of student engagement

This leads to inefficiency, confusion, and poor engagement.

---

## 💡 Solution

QUMO solves these problems by providing a **single unified platform** where:

- Students can participate in all campus activities
- Coordinators can manage events and engagement easily
- Faculty can monitor student involvement and progress

Everything is connected under one system.

## ⚙️ Tech Stack (Suggested)

- **Frontend:** React.js + Tailwind CSS  
- **Backend:** Node.js + Express.js  
- **Database:** PostgreSQL / Supabase  
- **Auth:** JWT / Supabase Auth  
- **Storage:** Cloud storage for files & certificates
